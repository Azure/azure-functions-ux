Develop

The fastest way to edit code is with the code editor, but you can also use Git. This example uses NodeJS, but many other languages are also supported.

This page also includes a log stream and a test console for helping you debug your function.

Integrate

You can easily integrate with other services and data sources from here.

  • Trigger - A function can be run based on an external action, such as when an item is inserted into a queue.
  • Inputs - Other, non-triggering data sources can be added and conditioned on the trigger data. These are passed in as parameters to your function.
  • Output - The results of your function's execution can be sent out to multiple targets.

Manage

This is the function management tab, where you will find configuration options specific to your function.

If you want to stop your function from triggering, you can disable it here. This is also where you can delete your function, so be careful!

Function App Settings

Your functions run within a "Function App." It houses centralized settings for your functions and gives you the ability to add features such as continuous integration and user authentication.

This screen is where you can configure settings which affect all of the functions within your function app.
